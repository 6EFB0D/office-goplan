# PictComp アセット

このフォルダには PictComp の画像アセットを配置します。

## スクリーンショット

| ファイル名 | 説明 | 撮影手順 |
|------------|------|----------|
| `gui-main.png` | メイン画面 | プリセット・圧縮設定・フォルダ選択・ファイル一覧・圧縮開始ボタンが見える状態で撮影 |
| `gui-compressed.png` | 圧縮完了 | プログレスバー完了、処理結果が表示されている状態で撮影 |
| `gui-viewer.png` | 画像ビューア | サムネイル一覧が表示されている状態で撮影 |

## 推しポイント画像

| ファイル名 | 推しポイント |
|------------|--------------|
| `point-1.png` | プリセットで用途別に最適化 |
| `point-2.png` | プレビューで確認してから実行 |
| `point-3.png` | サムネイル表示で画像を確認・操作 |
| `point-4.png` | EXIF情報の閲覧・管理 |
| `point-5.png` | フォルダ選択で一括処理 |
| `point-6.png` | 複数形式に対応 |
| `point-7.png` | 14日間全機能トライアル |
| `point-8.png` | 圧縮完了の結果表示 |

## 画像の差し替え

実画像を用意したら、同名の PNG で保存し、`pictcomp.html` の画像パスを `.svg` から `.png` に変更してください。
