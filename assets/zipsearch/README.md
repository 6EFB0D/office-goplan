# ZipSearch スクリーンショット

このフォルダには ZipSearch の UI キャプチャを配置します。

## 必要な画像

| ファイル名 | 説明 | 撮影手順 |
|------------|------|----------|
| `gui-main.png` | GUI メイン画面 | ZipSearch GUI を起動し、検索パス・キーワード・オプション・検索ボタンが見える状態で撮影 |
| `gui-results.png` | 検索結果画面 | 検索実行後、結果一覧が表示されている状態で撮影 |
| `web-main.png` | WebUI メイン画面 | Web 版を起動し、メイン画面で撮影（任意） |

## 撮影手順

1. ZipSearch を起動（GUI 版）
2. 検索パスにサンプル ZIP を追加
3. 検索キーワードを入力した状態で **gui-main.png** を撮影
4. 検索実行後、結果が表示された状態で **gui-results.png** を撮影
5. （任意）Web 版を起動し、メイン画面で **web-main.png** を撮影

## 画像の差し替え

スクリーンショットを更新する場合は、上記の手順で撮影し、同名で上書き保存してください。
