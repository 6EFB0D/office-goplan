# pdfHandler アセット

このフォルダには pdfHandler 製品ページ用の画像アセットを配置します。

## アイコン

- `PdfHandler.ico`: `pdf-handler/src/PdfHandler.UI/Assets/PdfHandler.ico` からコピーしたアプリ本体のアイコン。
- `PdfHandler.png`: Web表示用。`.ico` の外周につながる白背景を透明化したアイコン。

## スクリーンショット

- `gui-main.png`: メイン画面。フォルダツリー、PDF 一覧、右側プレビュー、上部ツールバーが見える状態。
- `gui-preview.png`: PDF プレビュー操作。ページ送り、拡大・縮小、選択中 PDF の内容が分かる状態。
- `merge-dialog.png`: PDFファイルの結合ダイアログ。結合順序、上へ/下へ、保存先、ファイル名が見える状態。
- `split-dialog.png`: PDFファイルの分割ダイアログ。ページ範囲、1ページずつ、等分割、保存先が見える状態。
- `page-operations.png`: ページの削除・挿入ダイアログ。削除ページ指定、挿入元 PDF、出力先が見える状態。
- `header-footer.png`: ヘッダ・フッターダイアログ。文書タイトル、ページ番号形式、フォント設定が見える状態。
