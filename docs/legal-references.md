# 法務ページ比較用リファレンス一覧

プライバシーポリシー・利用規約・特定商取引法に基づく表記（以下「特商法表記」）を、規模感の違う事業者の公開例と照らし合わせるためのメモです。

**あくまで公開資料の構造・粒度の参考**であり、自社にそのまま当てはまるかは事業内容・取引形態・個人情報の扱いにより異なります。最終判断は専門家への確認を推奨します。

---

## 1. 読み方（比較する時の軸）

| 観点 | 見る内容の例 |
|------|----------------|
| **プライバシー** | 取得するデータの列挙、第三者提供、Cookie・外部送信、保管期間、開示請求・問い合わせ窓口 |
| **利用規約** | サービス定義、禁止事項、免責、料金・更新、準拠法・管轄、規約改定 |
| **特商法** | 事業者名・所在地・連絡先、代金・支払・提供時期、返品・解約（デジタルは定型が難しいことが多い） |

大企業は **規約が細分化**（サービス別 URL、関連ポリシーが複数）されていることが多いです。同規模のソフト販売では **1 ページにまとまっている**例が多く、**ページ数で比べすぎない**方がよいです。

| 事業者（イメージ） | 法務・特商法の入口 |
|--------------------|-------------------|
| **株式会社 CAMPFIRE**（プラットフォーム） | [特定商取引法に基づく表記](https://camp-fire.jp/legal)（同一ページ付近に規約類がまとまっている構成の参考） |

## 4. 同規模〜やや小規模に近い例（個人・小規模開発）

「条文が薄く、取得データが少ない」など**現実的なボリューム**の参考。掲載場所は **自ドメイン / GitHub Pages / note** など様々です。

| タイプ | 例（プライバシー等） | メモ |
|--------|----------------------|------|
| 個人サイト | [tinylab-apps（プライバシーポリシー）](https://tinylab-apps.web.app/PrivacyPolicy/ja.html) | 英語併記など、アプリストア向けの簡潔な例 |
| 個人・小規模 | [Programmable / mechengjp](https://www.mechengjp.com/%E3%83%97%E3%83%A9%E3%82%A4%E3%83%90%E3%82%B7%E3%83%BC%E3%83%9D%E3%83%AA%E3%82%B7%E3%83%BC/) | 製品サイト直下にポリシー |
| 個人開発（アプリ） | [つみカロ / OJTHON.dev](https://ojthon-dev.com/company/tumi-cal/privacy-policy) | 製品単位で `/company/...` に分ける例 |

**特商法表記**は個人開発でも **Stripe 申込・自サイト販売**で必要になることが多いです。同規模の**直接の URL**は公開されていない・検索で揺れることがあるため、次を併用するとよいです。

- **手動検索**: 「アプリ名 + 特定商取引法」「屋号 + とくしょうほう」
- **App Store / Google Play** の販売者情報に載る URL

## 6. 比較検証用チェックリスト（簡易）

自社（Office Go Plan）と並べるときの例:

- [ ] 取得する個人情報の種類は、実装・お問い合わせフォームと一致しているか
- [ ] Cookie / 解析 / 決済（Stripe）に触れているか
- [ ] 利用規約で「ライセンスの範囲」「禁止事項」「免責」「料金・解約」がサービスに合っているか
- [ ] 特商法表記の**連絡手段**は実際に監視できるか（メールのみ等、方針と一致しているか）
- [ ] 最終更新日・改定手続きを書いているか
