# 見積書・請求書・領収書 Word テンプレート

Stripe 発行の請求書 PDF とトンマナを揃えた、**Office Go Plan** 用の Word 書式 3 種。
Stripe Dashboard に頼らずに書類を発行できるよう、`python-docx` で一括生成する構成です。

---

## 生成物

| ファイル | 用途 |
|----------|------|
| `template_quote.docx`   | **見積書**（法人向け、月末締め・翌月末払い前提） |
| `template_invoice.docx` | **請求書**（銀行振込案内ブロック付き） |
| `template_receipt.docx` | **領収書**（但し書き付き、入金確認後に発行） |

3 種に共通で以下を含めています。

- 左上にタイトル／右上に Office Go Plan ロゴ
- 見出し直下の緑アクセントライン
- 書類番号・発行日・期限等のメタ情報
- 請求先（左）／発行元（右）2 列ブロック
- 大見出しの金額＋一言サブノート
- 説明／数量／単価／金額の明細表
- 右寄せの 小計／合計／合計（強調）
- **免税事業者 注記**（「不課税」「T 番号を有していない」「仕入税額控除の経過措置」）
- フッター: `Office Go Plan ／ support@office-goplan.com`

請求書のみ、以下を追加で含みます。

- 銀行振込のご案内ブロック（金融機関／支店／種別／口座番号／口座名義）
- 利用規約 **第5条の2**（法人後払いの催告・一時停止・失効・遅延損害金 年 14.6%）への参照

## 差し替え運用（1 件ずつ書類を発行するとき）

1. `template_*.docx` を **コピー** して、日付つきファイル名に改名
   - 例: `quote_2026-04-30_acme.docx`
2. Word で開き、以下のプレースホルダを上書き
   - 書類番号（`Q-YYYYMMDD-0001` など）
   - 発行日・期限
   - 請求先の社名・住所
   - 明細行（説明／数量／単価／金額）
   - 金額（大見出しと右寄せ合計）
3. PDF 書き出し（Word → 名前を付けて保存 → PDF、または印刷 → Microsoft Print to PDF）
4. PDF を顧客メールに添付し、**Gmail で送信**（DKIM 署名＋Gmail サーバーの受信時刻が確定日付の代用エビデンスになるため、送信済みフォルダは削除しない）

## 金額・書類番号の付番ルール（運用メモ）

| 書類 | 接頭辞 | 例 |
|------|--------|------|
| 見積書 | `Q-` | `Q-20260430-0001` |
| 請求書 | `I-` | `I-20260430-0001` |
| 領収書 | `R-` | `R-20260430-0001` |

- 連番は **日次リセットしない**（月次または年次で連続）
- 同一案件では、見積→請求→領収で **末尾連番を揃える** と追跡しやすい
  - 例: `Q-20260430-0007` → `I-20260501-0007` → `R-20260515-0007`

## 注意事項

### 免税事業者の取扱い（2026-04 時点）

- 当事業者は **免税事業者**（T 番号なし）
- 書類上は「**不課税（免税事業者）**」の税込一本価格で表示
- 消費税額を別建てで表示・請求することはしない
- 取引先（特に法人）には **仕入税額控除の経過措置** を書面で案内
  - ～2026/09/30: 相当額の 80% が控除可
  - 2026/10/01 ～ 2029/09/30: 50%
  - 以降: 控除不可

将来、適格請求書発行事業者として登録した場合は、本テンプレートと利用規約 第5条の3 を改定すること。

### 法人後払い

- 支払期日 翌日 + 7 日 → 催告メール送信
- 催告 + 7 日 → ライセンス一時停止
- 支払期日 翌日 + 60 日 → ライセンス失効・契約解除
- 遅延損害金 **年 14.6%**（合意特約）
- 根拠: 利用規約 第5条の2（https://office-goplan.com/terms-of-service.html）

### 住所表記

特商法表記・利用規約・本テンプレートとも **「〒106-0032 東京都港区六本木2-1-19 S-Building 3F」** で統一する。
Stripe Dashboard の請求先住所も、アカウント再開時に `3F`（`3階` ではなく）に揃えること。

| 項目 | 値 |

| 支店 | にじ支店（支店番号 302） |

### 収入印紙

- 電子発行（PDF 送付）の領収書は **課税文書に該当しないため印紙不要**
- 紙出力して手渡し／郵送する場合は、**5 万円以上** で印紙貼付が必要（2022 年 4 月以降、5 万円未満は非課税）
