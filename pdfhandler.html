pdfHandler

PDFをまとめて管理・編集

ファイルサーバ上の PDF をプレビューしながら、リネーム・結合・分割・ページ編集を行える Windows デスクトップアプリです。

概要

フォルダツリー、PDF 一覧、プレビューを 3 ペインで表示。PDF を開き直さずに内容を確認し、F2 キーでのファイル名変更、複数 PDF の結合、ページ範囲指定による分割、ページ削除・挿入、回転、ヘッダ・フッター追加までを一つの画面から操作できます。

PDFプレビュー / サムネイル表示 / リネーム / 結合 / 分割 / ページ編集 / 回転 / ヘッダ・フッター / お気に入りフォルダ

機能・特徴

  • 3ペイン構成: フォルダツリー、PDF 一覧、プレビューを一画面に表示
  • PDFプレビュー: サムネイル/リスト表示とページ送り、拡大・縮小に対応
  • ファイル操作: F2 キーでのリネーム、コピー、ペースト、削除を実行
  • PDF結合: 複数 PDF の順序を調整して 1 つの PDF に結合
  • PDF分割: ページ範囲、1ページずつ、等分割の 3 パターンに対応
  • ページ編集: ページ削除・挿入、現在ページまたは全ページの回転が可能
  • 文書整備: ヘッダ・フッター、ページ番号、PDF バインダー管理を搭載
  • ライセンス管理: アプリ内購入、ライセンスキー入力、デバイス管理に対応

スクリーンショット

メイン画面

フォルダツリー、PDF 一覧、右側プレビュー、上部ツールバーを一画面で確認

PDFプレビュー

ページ送り、拡大・縮小を使いながら選択中の PDF を確認

PDF結合

結合する PDF の順序を調整し、保存先とファイル名を指定

PDF分割

ページ範囲、1ページずつ、等分割の方法を選択して出力

ページ編集

ページ削除・挿入、挿入元 PDF、出力先を指定

ヘッダ・フッター

文書タイトル、ページ番号形式、フォント設定を指定

ダウンロード

Windows 版の公開準備を進めています。リリースページとダウンロードリンクは準備が整い次第ご案内します。
